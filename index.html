Wordle Analyzer

Do you know the game Wordle (https://wordle.danielfrg.com/)?

Wordle is a game where you should find a 5-letter word.
Every attempt should be a valid a 5-letter word (validated using the dictionary).
After each attempt the letters change color showing if the letter is on the final word and if it is in the correct place.

This game become a sensation among friends and we all want to win in the less amount of attempts as possible.

That's why you will apply some Code Magic to get us the best words to start the game.

We consider the best word to start the game the one with the most common letters among all the words of the dictionary.

Dictionary: 'dictionary.ts' contains a list of the words of the dictionary

Write some code to get the best word of the given dictionary to start the game with.

Tip: think about an score system or weight system to rank the words.
